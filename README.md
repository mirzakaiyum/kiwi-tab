## ğŸ§‘â€ğŸ’» Running Locally

### Prerequisites

- [Node.js](https://nodejs.org/) (Latest LTS)
- [npm](https://www.npmjs.com/)

### Run locally

- Clone this repository
- Install dependencies
- Start development server

```bash
# Clone the repo
git clone https://github.com/MakFissh/kiwi-tab.git

# Install dependencies
npm install

# Run dev server
npm run dev

# Build for extension
npm run build
```

#### Chrome/Edge

- Go to `chrome://extensions` or `edge://extensions`
- Enable Developer mode
- Load unpacked and select the `dist` folder after running `npm run build`.
